# NewBie-Plan
## 1. 概述
```
   主要针对于现阶段Java开发工程师开启的学习推荐和辅助，主要会从Java基础(IO,Collection集合,反射,多态),MySQL,多线程,JVM,操作系统，计算机网络及部分高级应用等方面做较为详细的学习探究。
   现阶段该项目由5个人持续维护,同时也希望有兴趣的同学能够加入我们,学习总结,一起成长!
```
```
  该项目持续维护周期为半年,请各位尽自己力量维护下知识库!
```
